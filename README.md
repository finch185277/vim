# vim
configure vim

## Files
### basic.vim
Without plugins
### vimrc.vim
Build with plugins

## Install
1. Download Vundle
  - `git clone https://github.com/gmarik/Vundle.vim ~/.vim/bundle/Vundle.vim`
2. Install Powerline fonts
  - `apt install fonts-powerline`
  - https://github.com/powerline/fonts
3. Move `vimrc.vim` to `~/.vimrc`
4. `vim +PluginInstall +qall`
5. Done!
